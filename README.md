# bent-comics
Comic Rent app example built with laravel v5.8.34 + react.js

## Getting Started
### Installing
```
composer install
```
migrate database after setting up .env file
```
composer run migrate
```
install react.js
```
npm install
```

### Usage
```
npm start
```
